🐙 How to Put TREASURE on GitHub
Follow these steps to upload your app so the whole world can use it
01
Download This File
First, download the index.html file from this page (the file you are using right now). Save it somewhere easy to find like your Desktop.
⚠️ Make sure the filename is exactly index.html — not TREASURE.html
02
Log in to GitHub
Go to github.com and log in with your account FilmKeeper-1
03
Create a New Repository
Click the + icon (top right) → New repository
Fill in like this:
Repository name: treasure Visibility: ✅ Public Add README: ✅ Check this box
Then click the green "Create repository" button
04
Upload Your index.html File
Inside your new repository:
1. Click "Add file" → "Upload files" 2. Drag your index.html file onto the page 3. Scroll down → Click "Commit changes" 4. Click "Commit changes" again to confirm
⚠️ The file MUST be named index.html or the site will show a blank white page
05
Turn on GitHub Pages
This makes your site live on the internet:
1. Click "Settings" (top menu of your repo) 2. Left sidebar → Click "Pages" 3. Under Source → select "Deploy from branch" 4. Branch → select "main" → folder "/ (root)" 5. Click "Save"
06
Wait 2 Minutes Then Visit Your Site
Refresh the Pages settings page and you will see a green box saying your site is live at:
https://filmkeeper-1.github.io/treasure
✅ Share this link with anyone and they can use TREASURE on their phone or computer!
07
Need Help? Contact Us
